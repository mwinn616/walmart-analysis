# Walmart Sales Analysis

Exploratory analysis of weekly sales data across 45 Walmart stores from 2010–2012.

## Tools
- Python, pandas, matplotlib
- SQL (SQLite)
- Jupyter Notebook

## Key Findings
- Holiday weeks drive 7.8% higher average weekly sales
- Store 20 is the top performer at ~$2.1M average weekly sales
- Sales show consistent seasonality with peaks in November and December each year
- 45 stores analyzed, zero missing data points
